## Commands to Run the project 

**Start Truffle**
```
truffle develop
```

**Run Tests**
```
truffle test
```

**Run Migrations with Infura**
```
truffle migrate --network ropsten_infura
```
**Install Frontend Dependencies**
```
cd client/
npm install
```

**Run Frontend**
```
cd client/
npm run start
```
